## 📥 DESCARGAR TODAS LAS IMÁGENES

### Opción 1: Script Python (Recomendado)

```bash
# Desde el root del proyecto
python descargar_imagenes.py

# Verificar descargas
python descargar_imagenes.py verify
```

**Esto descargará todas las imágenes a:**
```
desktop/assets/exercises/fixed/{nombre}.png
```

### Opción 2: Descarga Manual

Puedes descargar las imágenes una por una haciendo clic en los enlaces de arriba, o usar `curl`:

```bash
cd desktop/assets/exercises/fixed

# Ejemplo: descargar bench press
curl -O "https://raw.githubusercontent.com/yuhonas/free-exercise-db/main/exercises/barbell%20bench%20press.png"

# Renombrar
mv "barbell%20bench%20press.png" "barbell_bench_press.png"
```

### Opción 3: Clonar Repositorio Completo

```bash
git clone https://github.com/yuhonas/free-exercise-db.git
cp free-exercise-db/exercises/*.png desktop/assets/exercises/fixed/
```

## 📱 CÓMO SE USAN LAS IMÁGENES

### Mobile (React Native)

**Ejercicios predefinidos:**
```typescript
// En exercises_catalog.ts
image_url: "https://raw.githubusercontent.com/.../barbell%20bench%20press.png"
```

Las imágenes se cargan directamente desde la URL. React Native cachea automáticamente las imágenes.

**Ejercicios custom:**
```typescript
// El usuario selecciona una imagen local
// Se copia a: {DocumentDirectory}/custom_exercises/{id}.jpg
image_uri: "file:///data/.../custom_exercises/abc123.jpg"
```

### Desktop (Python Tkinter)

**Ejercicios predefinidos:**
```python
# En db.py, al hacer seed
image_uri = "https://raw.githubusercontent.com/.../barbell%20bench%20press.png"

# Para mostrar en UI, descargar localmente primero:
image_path = "assets/exercises/fixed/barbell_bench_press.png"
```

**Ejercicios custom:**
```python
# Usuario selecciona imagen local vía file dialog
# Se copia a: assets/exercises/custom/{id}.jpg
image_uri = "file://./assets/exercises/custom/abc123.jpg"
```

## 🔄 ACTUALIZAR CATÁLOGO DE IMÁGENES

Si quieres agregar más ejercicios:

1. **Busca el ejercicio en ExerciseDB:**
   - Ve a https://github.com/yuhonas/free-exercise-db/tree/main/exercises
   - Busca el archivo JSON del ejercicio
   - Copia el nombre exacto

2. **Agrega al catálogo:**
   - Edita `shared/constants/exercises_catalog.ts`
   - Agrega un nuevo objeto al array `EXERCISE_CATALOG`

3. **Descarga la imagen:**
   ```bash
   python descargar_imagenes.py
   ```

## 📝 PROBLEMAS COMUNES

### Las imágenes no cargan en Mobile

**Causa:** El dispositivo no tiene conexión a internet.

**Solución:**
1. Conecta el dispositivo a WiFi
2. Abre la app para que carguen las imágenes
3. React Native las cacheará automáticamente
4. Ya no necesitarás internet

### Las imágenes no se muestran en Desktop

**Causa:** No has descargado las imágenes localmente.

**Solución:**
```bash
python descargar_imagenes.py
```

### Error 404 al descargar

**Causa:** El nombre del ejercicio tiene caracteres especiales.

**Solución:**
- Verifica el nombre exacto en el repositorio
- Usa `%20` para espacios en URLs
- Actualiza `descargar_imagenes.py` con el nombre correcto
